# GraphQL Postgres Express demo

## Setup

Install and use node 6 (if it's not already installed):

```
nvm install 6
nvm use 6
```

Set up the database:

```
psql -f setup_database.sql
```

## Running the app

```
npm run start
```

Then, go to [localhost:3000/graphql](http://localhost:3000/graphql).
